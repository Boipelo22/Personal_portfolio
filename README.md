# Boipelo Masenya Portfolio

A personal portfolio website for Boipelo Masenya showcasing skills, projects, contact details, and an interactive one-page layout.

## Overview

This portfolio is built using standard web technologies: **HTML5**, **CSS3**, and **JavaScript**. It includes animated scrolling, a responsive navigation menu, and dynamic typing text effects to create a polished and modern presentation of personal branding.

## Features

- Responsive one-page layout with fixed header navigation
- Animated scroll reveal effects for content sections
- Dynamic typed text effect for title emphasis
- Active section link highlighting while scrolling
- Smooth mobile menu toggle
- Sections included:
  - Home
  - About
  - Skills
  - Projects
  - Contact

## Tech Stack

- HTML
- CSS
- JavaScript
- Google Fonts: Poppins
- Boxicons icon library
- Typed.js for typing animation
- ScrollReveal for scroll-based animations

## Installation

1. Clone or download the repository.
2. Ensure the following files are present in the root folder:
   - `index.html`
   - `style.css`
   - `javascript.js` (or rename to `script.js` to match `index.html`)
   - `images/` folder containing profile images
3. Open `index.html` in a web browser.

## Usage

- Navigate the page using the header links.
- Click the hamburger menu on smaller screens to open or close navigation.
- Use the contact form to send a message (client-side form only).
- Replace the placeholder images in `images/` with your own photos for customization.

## Project Structure

- `index.html` - main website structure and content
- `style.css` - styling, layout, and responsive behavior
- `javascript.js` - interactive behaviors and animations
- `images/` - media assets used in the portfolio

## Notes

- `index.html` currently references `script.js` for the page script. If your JavaScript file is named `javascript.js`, either rename the file to `script.js` or update the `src` attribute in `index.html`.
- The contact form is static and does not include backend submission handling.

## Customization

To personalize this portfolio:

- Update the name, introduction, and about text in `index.html`
- Replace the social links with your own profiles
- Add or modify project cards in the Projects section
- Tweak colors and spacing in `style.css`
- Replace the profile images in the `images/` folder
